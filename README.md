Talk to your coding agents.

Hook development lives in [docs/codex-hooks.md](docs/codex-hooks.md). The hook layer turns Codex lifecycle events into a stable local event contract for the Code Talker app and GPT-realtime-2 service.

The Swift service API for the UI/realtime teams is documented in [docs/code-talker-service.md](docs/code-talker-service.md).
